Velopack is used in this project.

prerequisites;
dotnet tool install -g vpk
.

TO BUILD;

1. 
CHANGE VERSION IN CSPROJ

2.
PUBLISH (SELF CONTAINED)

3.
vpk pack `
--packId MySchool `
--packAuthors HSP-Studios `
--packDir MySchool\bin\Publish `
--mainExe MySchool.exe `
--icon MySchool\resources\logo\ico\Dark-Icon.ico `
--splashImage MySchool\resources\logo\png\Dark-Wide.png `
--packVersion VERSION 

4. UPLOAD TO GITHUB
- assets.win.json
- releases.win.json
- MySchool-win-Setup.exe
- MySchool-win-Portable.zip
- MySchool-v*-full.nupkg
- MySchool-v*-delta.nupkg

5. UPLOAD TO S3 (Fallback for update checks)
Upload the same files to the S3 bucket: myschool-ap-se-2.s3.ap-southeast-2.amazonaws.com
- assets.win.json
- releases.win.json
- MySchool-v*-full.nupkg
- MySchool-v*-delta.nupkg

Note: The app will first try GitHub for updates (10 second timeout). 
If GitHub is unreachable or slow, it automatically falls back to the S3 bucket.
